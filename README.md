# Welcome to GameHead!

	> A central location to get the latest Games that are out. Made using  **Synchronize now** in Android Studio
	> Before starting to sync files, you must link an account in the **Synchronize** sub-menu.


# Components

**1. Kotlin** 
**2. Jetpack Compose**                          
**3. MVVM Architecture**
**4. Retrofit**
**5. Dagger Hilt**
**6. FreeToGame API**
